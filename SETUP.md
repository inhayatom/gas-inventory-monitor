# セットアップ手順

## 1. スプレッドシート準備

### シート構成
- データシート
- 設定シート
- ログシート

### データシートの列
| A列 | B列 | C列 | D列 | E列 |
|-----|-----|-----|-----|-----|
| 商品名 | 在庫数 | 最終チェック | 納期 | ステータス |

### 設定シート
| A列 | B列 |
|-----|-----|
| 監視対象列 | B |
| 在庫アラート閾値 | 10 |
| データ開始行 | 2 |
| LINE Channel Access Token | （トークン） |
| 納期列 | D |
| 納期アラート日数 | 7 |
| ステータス列 | E |
| 監視対象ステータス | 未対応 |

## 2. LINE Messaging API設定

1. [LINE Developers](https://developers.line.biz/)にアクセス
2. Messaging APIチャネル作成
3. Channel Access Tokenを発行
4. Botを友だち追加
5. 設定シートにトークンを貼り付け

## 3. GASコード設定

1. スプレッドシートで「拡張機能」→「Apps Script」
2. `Code.gs`の内容を全てコピー＆ペースト
3. 保存

## 4. トリガー設定

1. Apps Script画面で「トリガー」をクリック
2. またはスプレッドシートのメニュー「📊在庫監視」→「⚙️トリガー設定」

## 5. 動作確認

データシートのB列を編集して、LINE通知が届くか確認

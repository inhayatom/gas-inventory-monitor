# 📊 GAS在庫監視システム

Google Apps Scriptで構築した、スプレッドシートの自動監視＆LINE通知システムです。

## ✨ 機能

- ✅ **複数条件監視**: 在庫数、納期、ステータスを同時監視
- ✅ **LINE通知**: アラート発生時に自動でLINE通知
- ✅ **自動実行**: 1時間ごと + 編集時の即時チェック
- ✅ **ログ記録**: 実行履歴を自動記録
- ✅ **設定シート**: コード変更不要で条件変更可能

## 🎯 ユースケース

### 汎用
- タスク管理（期限監視）
- 承認フロー（未対応アラート）
- 日報管理

### EC・物販
- 在庫アラート
- 発送期限監視
- 受注状況管理

### 教育・塾
- 月謝未納アラート
- 入退室管理
- 体験授業予約管理

## 🔧 セットアップ

### 必要なもの
- Googleアカウント
- LINE Messaging APIアカウント（無料）

### 手順
1. Googleスプレッドシートを作成
2. 「拡張機能」→「Apps Script」を開く
3. このリポジトリのコードをコピー＆ペースト
4. LINE Messaging APIのトークンを設定
5. トリガーを設定

詳細な手順は[こちら](設定手順書のリンク)
